Respuestas a Preguntas de JavaScript

1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias:

  • Java es un lenguaje de programación de propósito general y compilado, mientras que JavaScript es un lenguaje de programación interpretado, principalmente utilizado en desarrollo web.
  • Java es más complejo en cuanto a sintaxis y configuración, mientras que JavaScript es más flexible y mucho mucho mucho más dinámico.

Semejanzas:

  • Ambos son lenguajes de programación basados en C.
  • Ambos permiten la programación orientada a objetos (aunque JavaScript lo hace de forma más flexible y no estricta) lo cual es mucho mejor para mi, ya que la progrmación orientada a objetos siempre es un poco especial.

2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • getDate()
  • getMonth()
  • getFullYear()
  • getHours()
  • getMinutes()

3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • push()
  • pop()
  • shift()
  • unshift()
  • splice()

4. ¿Cómo se declara una variable con alcance local dentro de una función?

Se declara utilizando let o const dentro de la función.

5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Las variables globales pueden ser modificadas por cualquier parte del código, lo que puede generar efectos secundarios inesperados, dificultando el mantenimiento del código y aumentando el riesgo de errores.

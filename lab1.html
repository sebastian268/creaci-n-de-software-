¿Cuál es la diferencia entre Internet y la World Wide Web?

La Diferencia entre Internet y la World Wide Web es que el internet es la red global a la que se conectan todos los dispositivos para poder comunicarse entre ellos; Y la World Wide Web es un servicio que existe dentro del mismo internet que permite acceder a paginas web que están en el innternet.

¿Cuáles son las partes de un URL?

Hay varias partes que conforman una URL, dentro de las partes que conforman la URL se encuentran:

- El protocolo que se utiliza para la creación de la página web (normalmente http)

- El dominio, el cual normalmente se ve identificado como www.ejemplo.com

Estas dos primeras, son idispensables para cualquier URL, aunque tambien pueden tener lo siguiente:

- La ruta a seguir

- Parámetros.

- Fragmento.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

El propósito de los métodos HTTP es el siguiente:

GET: Su propósito es el obtener datos e información pero sin modificarlos, solo para visualizarlos.

HEAD: Este es un método que tiene un funcionamiento similar a GET, pero este sirve solo para devolver encabezados.

POST: Este método sirve para enviar datos al servidor, es decir, para crear nuevos recursos dentro del servidor.

PUT: Este método sirve para modificar, o en su defecto remplazar un recurso dentro del servidor.

PATCH: Este es un método que se utiliza para modificar de manera temporal un recurso, normalmente se utiliza para arreglar algún error de manera temporal.

DELETE: Este método sirve para eliminar un recurso del servidor.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

Para enviar un formulario el método que se utiliza es el de POST debido a que los datos de este tipo son datos de una sensibilidad mayor, y de esta manera los datos se envian en el cuerpo y no en la URL

El método que se utiliza para acceder a una página web mediante el uso de una URl es el de GET, ya que de esta manera el navegador Web solicita al servidor los datos de la página

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

Cuando el servidor web devuelve una respuesta HTTP al usuario con un código 200 significa que no ocurrió ningún error, al contrario, el servidor est devolviendo un "OK"

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Yo considero que no es necesario que el desarrollador corriga el error, debido a que el error 404 significa "not found" significa que no se ha podido encontrar la página web por lo que podria significar que el usuario esta ingresando de manera incorrecta la URL

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

En este caso si es responsabilidad del desarrollador debido a que un error 500 significa "Internal Server Error" lo cual quiere decir que el error se encuentra dentro servidor, o en su defecto en el código base,

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

La diferencia principal entre HTML4 Y HTML5 es el uso de las etiquetas, ya que HTML4 tiene etiquetas que son consideras genéricas como lo es "div" y al contrario HTML5 tiene uso de etiquetas semánticas las cuales describen de una manera mucho mejor el contenido

¿Qué componentes de estructura y estilo tiene una tabla?

los elementos que definen a una tablas son los siguientes:
- table: sirve para crear y definir la tabla
- tr: Define una fila.
- th: Celda de encabezado.
- td: Celda de datos.
Tambien existen algunas etiquetas que son opcionales como lo son "caption", "thead", "tbody"

¿Cuáles son los principales controles de una forma HTML5?

Los principales controles de una forma en HTML5 son los siguientes:
- "form": Permite enviar datos a un servidor.
- "input": para ingresar datos, además cuenta con el atributo "type" el cual cambia su comportamiento
- "textarea": Crea un área de texto grande
- "select": Muestra una lista de opciones.
- "button": Crea un botón que puede enviar el formulario o realizar una acción.

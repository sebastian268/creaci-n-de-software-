¿Cuál es la diferencia entre Internet y la World Wide Web?

La diferencia entre Internet y la World Wide Web es que el Internet es la red global a la que se conectan todos los dispositivos para poder comunicarse entre ellos, mientras que la World Wide Web es un servicio que existe dentro del mismo Internet y permite acceder a páginas web.

¿Cuáles son las partes de un URL?

Hay varias partes que conforman una URL, dentro de las cuales se encuentran:

  • El protocolo (normalmente http)
  • El dominio (normalmente www.ejemplo.com)
  • La ruta a seguir
  • Parámetros
  • Fragmento

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

El propósito de los métodos HTTP es el siguiente:

  • GET: Su propósito es obtener datos e información, pero sin modificarlos.
  • HEAD: Similar a GET, pero solo devuelve los encabezados.
  • POST: Envia datos al servidor para crear nuevos recursos.
  • PUT: Modifica o reemplaza un recurso en el servidor.
  • PATCH: Modifica temporalmente un recurso, normalmente para corregir errores.
  • DELETE: Elimina un recurso del servidor.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

El método que se utiliza es el POST porque los datos que se envían (como usuario y contraseña) son sensibles, y el POST permite que se envíen en el cuerpo de la solicitud en lugar de en la URL.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

El método utilizado es el GET, ya que el navegador solicita al servidor los datos de la página.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

La diferencia principal entre HTML4 Y HTML5 es el uso de las etiquetas, ya que HTML4 tiene etiquetas que son consideras genéricas como lo es "div" y al contrario HTML5 tiene uso de etiquetas semánticas las cuales describen de una manera mucho mejor el contenido

¿Qué componentes de estructura y estilo tiene una tabla?

Los elementos que definen a una tabla son los siguientes:

  • table: sirve para crear y definir la tabla.
  • tr: Define una fila.
  • th: Celda de encabezado.
  • td: Celda de datos.

También existen algunas etiquetas que son opcionales como lo son:

  • caption: Define un título para la tabla.
  • thead: Define el encabezado de la tabla.
  • tbody: Define el cuerpo de la tabla.

¿Cuáles son los principales controles de una forma HTML5?

Los principales controles de una forma en HTML5 son los siguientes:

  • form: Permite enviar datos a un servidor.
  • input: Para ingresar datos, además cuenta con el atributo "type" el cual cambia su comportamiento.
  • textarea: Crea un área de texto grande.
  • select: Muestra una lista de opciones.
  • button: Crea un botón que puede enviar el formulario o realizar una acción.

¿Cuál es tu recomendación al elegir entre %, px y pt en CSS?

Usar % para diseños adaptables (responsivos). Utilizar px para medidas exactas o detalles específicos. Evitar pt, ya que está pensado para impresión.
